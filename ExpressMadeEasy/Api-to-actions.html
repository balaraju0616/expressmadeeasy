KeshavSoft Express API Journey

From Empty Folder → API Actions Generation using KeshavSoft VS Code Extensions

STEP 1 — Create Empty Folder

Create an empty folder:

test

Open the folder in VS Code.

STEP 2 — Install Required Extensions

Install these VS Code extensions:

  • ks-express-boilerplate
  • express api generator
  • express api sub routes generate
  • express api endpoints generate
  • express api actions generate

After installing all extensions, restart VS Code once.

STEP 3 — Generate Express Boilerplate

Right click on empty Explorer area.

Select:

Express BoilerPlate with simple

This generates:

app.js
routes.js
server.js
Config/
Public/
package.json

STEP 4 — Generate Api Folder

Open: app.js

Click the button below to view the complete process.

Open Step 1

STEP 5 — Generate V1 Sub Route

Open: Api/routes.js

Click the button below to view the complete process.

Open Step 2

STEP 6 — Generate Table Route

Open: Api/V1/routes.js

Click the button below to view the complete process.

Open Step 3

STEP 7 — Generate API Actions

Open: Api/V1/journals/end-points.js

Use:

get
insert
delete
alter

This generates API action folders like:

ShowAll/
Insert/
Delete/
Alter/

Along with:

controller.js
service.js

rest.http
Open Step 4
